# WEBSITE BÁN ĐIỆN THOẠI PHONESHOP

Xây dựng website bán điện thoại cho cửa hàng PhoneShop.

## Thành viên thực hiện

-   Trần Văn Hùng (Leader).
-   Trương Văn Toàn.

## Công nghệ sử dụng

-   **Front-end:** ReactJS, Redux (Toolkit), Bootstrap.
-   **Back-end:** Nodejs.
-   **CSDL:** MongoDB.

## Hướng dẫn chạy

### Front-end

```
cd client

yarn start
```

### Back-end

```
cd server

npm run server
```
