# WEBSITE BÁN ĐIỆN THOẠI PHONESHOP
Xây dựng website bán điện thoại cho cửa hàng PhoneShop.

## 1. Thành viên thực hiện
- Trần Văn Hùng (Leader).
- Trương Văn Toàn.

## 2. Công nghệ sử dụng
- **Front-end:** ReactJS, Redux (Toolkit), Bootstrap.
- **Back-end:** Nodejs.
- **CSDL:** MongoDB.

## 3. Hướng dẫn cài đặt
### 3.1. Front-end
``
cd client
yarn start
``

### 3.1. Back-end
``
cd server
npm run server
``
